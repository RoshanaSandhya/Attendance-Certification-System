# Attendance Certification System

## Project Overview

This project automates attendance tracking for 980 students across 40 classes. The system loads daily attendance sheets, calculates attendance percentages, and identifies students eligible for certification.

## Features

* Load attendance data from multiple Excel/CSV files
* Calculate attendance percentage automatically
* Identify certified and non-certified students
* Generate attendance reports
* Machine Learning based classification using Decision Tree

## Certification Rule

Students with attendance greater than or equal to 80% are marked as Certified.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Jupyter Notebook

## Output

* Student Name
* Classes Attended
* Attendance Percentage
* Certification Status
